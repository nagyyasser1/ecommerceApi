e-Shop api

Swagger Docs

USER

  • Create an account
  • Login/logout/verify-email
  • Add an address
  • View all products (featured/filtering/search)
  • Write/update/delete a review
  • Place an order
  • View his orders
  • Cancel an order (if status is shipped, it can't be canceled)
  • Create a wishlist / Get own wishlist / Delete it
  • Add / update / delete / get items to card
  • Contact e-shop owner

ADMIN

  • Can do everything an ordinary user can
  • Delete/update a user
  • Add/modify/delete a product
  • Delete any review
  • Cancel orders
  • View all orders
